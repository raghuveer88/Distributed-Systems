This is a one-one communation between N clients using a server program.

For code excution commands-
python3 server.py (server will start running)
python3 client.py (client will start running)

then you can run client program multiple times to connect new clients
once clients are connected you can communicate with each other using the index of the client and "-" message
EX - to message person 1(receiver) from person 0(sender)
1-hello

the corresponding output on receviers side is 
user0: hello
